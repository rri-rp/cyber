# Cybersecurity Login Prototype

## Description

This is an early prototype for my final project. It is a JavaScript-based login demo that introduces basic cybersecurity concepts such as password strength, brute-force attacks, and XSS safety.

## Features

- Registration form
- Login form
- Failed login attempt tracking
- Password strength checker
- Brute-force simulation
- Basic XSS safety demo

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open the `index.html` file in a web browser.

## Progress

So far, I have created the main prototype page and added JavaScript for registration, login, password checking, failed attempts, and simple security demos.
